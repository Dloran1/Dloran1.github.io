VPN na Linux — konfiguracja krok po kroku (2025)

Linux daje duże możliwości, ale konfiguracja VPN wymaga kilku kroków. Poniżej znajdziesz prosty przewodnik dla Ubuntu, Debian, Fedora oraz Arch — zarówno dla OpenVPN, jak i WireGuard.

Konfiguracja OpenVPN

  1. Zainstaluj klienta: sudo apt install openvpn (Ubuntu/Debian) lub sudo dnf install openvpn (Fedora).
  2. Pobierz plik .ovpn od dostawcy VPN.
  3. Połącz się: sudo openvpn --config plik.ovpn.

Więcej o stabilności znajdziesz w VPN z Kill Switch.

Konfiguracja WireGuard

  1. Zainstaluj: sudo apt install wireguard lub sudo dnf install wireguard-tools.
  2. Otrzymasz plik wg0.conf z kluczami i serwerem.
  3. Uruchom: sudo wg-quick up wg0. Aby zatrzymać: sudo wg-quick down wg0.

Porównanie protokołów znajdziesz w WireGuard vs OpenVPN.

NetworkManager GUI

Jeśli wolisz interfejs graficzny (Ubuntu, Fedora Workstation):

  1. Otwórz Ustawienia sieci → Dodaj VPN.
  2. Zaimportuj plik .ovpn lub .conf.
  3. Zapisz i kliknij „Połącz”.

Podobne rozwiązanie działa też na iPhone czy Androidzie.

Test połączenia (IP/DNS)

  1. Sprawdź IP na ipleak.net.
  2. Jeśli widzisz IP VPN — działa poprawnie.
  3. Sprawdź też test wycieków DNS.
