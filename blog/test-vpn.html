Jak sprawdzić, czy VPN działa poprawnie

1) Adres IP

Wejdź na dowolny checker IP i upewnij się, że lokalizacja to **Polska** po połączeniu z serwerem PL.

Przykładowe narzędzia: whatismyipaddress.com, ipinfo.io

2) Wycieki DNS

Uruchom test DNS leak – wszystkie serwery powinny wskazywać Polskę lub dostawcę VPN, nie Twojego operatora.

3) WebRTC

Sprawdź, czy WebRTC nie ujawnia lokalnego IP. W razie potrzeby wyłącz WebRTC w przeglądarce lub użyj dodatku blokującego.

4) Prędkość

Wykonaj test speed (np. speedtest.net). Porównaj z/bez VPN i ewentualnie zmień serwer lub protokół dla lepszego wyniku.

Najczęstsze wnioski

Jeżeli IP/DNS wskazują inny kraj – przełącz serwer lub wyczyść cache DNS (restart urządzenia pomaga).
Słaba prędkość – wybierz bliższy serwer PL lub inny protokół (często WireGuard jest szybszy).
