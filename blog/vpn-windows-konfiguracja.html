Konfiguracja VPN w Windows 10/11 — poradnik krok po kroku

Windows 10 i 11 posiadają wbudowaną obsługę VPN, a także wspierają aplikacje zewnętrzne jak NordVPN czy Surfshark. Poniżej znajdziesz instrukcję krok po kroku, jak dodać VPN w Windows oraz jak sprawdzić, czy działa poprawnie.

Szybki start

Pobierz NordVPN na Windows Test prędkości VPN

Link partnerski — bez dodatkowych kosztów dla Ciebie.

1) Dodanie VPN w ustawieniach Windows

  1. Otwórz Ustawienia → Sieć i Internet → VPN.
  2. Kliknij Dodaj połączenie VPN.
  3. Dostawca VPN: Windows (wbudowane).
  4. Typ: L2TP/IPsec lub IKEv2 (zalecane).
  5. Adres serwera, login i hasło otrzymasz od dostawcy VPN.

2) Import pliku OpenVPN (.ovpn)

Wielu dostawców (np. NordVPN, ProtonVPN) udostępnia pliki konfiguracyjne. Do ich użycia zainstaluj klienta OpenVPN GUI i zaimportuj plik .ovpn.

3) Aplikacje dostawcy VPN

  • Pobierz oficjalną aplikację (np. NordVPN).
  • Zaloguj się, wybierz serwer w PL lub UE.
  • Włącz opcje: Kill Switch, DNS Leak Protection.

4) Testy i bezpieczeństwo

  • Sprawdź IP na stronie whatismyipaddress.com.
  • Wykonaj test wycieków DNS.
  • Porównaj prędkość: speedtest VPN.
